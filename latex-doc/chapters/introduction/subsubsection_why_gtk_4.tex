\subsubsection{Why GTK 4}

\gls{gtk} is cross-platform and looks modern. Compare to \gls{wxwidgets}, which was used originally for this project, that uses native API for each supported platform, \gls{gtk} is consistent in how it looks, also it supports 3D rendering more effectively.
